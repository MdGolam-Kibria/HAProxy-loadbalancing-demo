# loadbalancing-demo

#### Build:
`mvn clean install`

#### Running:
`java -jar -Dserver.port=8060 target/loadbalancing-demo-0.0.1-SNAPSHOT.jar`

#### Package and run:
This will build also  
`mvn package && java -jar -Dserver.port=8090 target/loadbalancing-demo-0.0.1-SNAPSHOT.jar`
